Standard Modules¶

A range of standard modules that can be used for versatile use cases.

Data Import and Export¶

  • GDALHotStarter Hot start simulation
  • GDALImportData Import data
  • GDALPublishResults Export GIS data

Linking¶

  • GDALSpatialLinking Spatially link data
  • NonSpatialLinking Link data sets with identifier
  • GDALJoinNearestNeighbour Link nearest neighbour
  • GDALCreateNeighbourhoodTable Link Adjacent Faces

Data Handling¶

  • GDALCopyElementToView Copy features
  • GDALRemoveComponets Delete components
  • GDALAttributeCalculator Calculate attributes
  • GDALGeometricAttributes Calculate area
  • GDALCalculateLength Calculate length
  • DM_CalculateDistance Calculate distance

Geometry Processing¶

  • GDALCreateCentroids Create centroids
  • GDALErase Subtract features
  • GDALOffset Offset faces

Groups¶

  • LoopGroup Repeatedly execute modules in the group to iteratively manipulate data

Urban From¶

  • GDALParceling Parcel faces
  • GDALParcelSplit Subdivide parcel
  • GDALCreateBuilding Generate simple buildings

Infrastructure Networks¶

A range of modules to help with data cleaning to prepare and analysis urban infrastructure networks. E.g. modules to integrate urban drainage networks into SWMM.

Data Preparation¶

  • GDALExtractNodes Creates start and end nodes for a given drainage network
  • GDALClusterNetwork Identify clusters of connected lines
  • DM_DirectGraph Direct graph
  • DM_ValueFromRaster Copy raster data values onto nodes

Network Analysis¶

  • DM_Strahler Strahler ordering
  • Segmentation Create segments of equal attributes
  • Dual Mapping Export dual mapped edge list

SWMM¶

  • GDALDMSWMM SWMM integration
  • DM_ImportSWMM Import SWMM File
