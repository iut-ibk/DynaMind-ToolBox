Mixer¶

Mixer combines two or more. Depending on the num_inputs parameter n inports are created. The ports are number from 0 to n-1.

Flow

f_{out,0}= \sum f_{in,0}

Concentrations

f_{out,i} = \frac{\sum f_{in,i} \cdot f_{in,0} } { \sum f_{in,0}} \\ i = 1 \dotsc n

Parameter¶

Name Type Description
num_inputs Integer Number of in ports default is 2
